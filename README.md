# Docker gRPC Health Check Example

## Requirements

- mise

## Setup

```sh
mise init
```

## Local

### Start Server

```sh
mise start
```

### Check health

```sh
mise check
```

## Container

### Build Image

```sh
mise container:build
```

### Start container

```sh
mise container:run
```

### Check with docker exec command

```sh
mise container:check
```
